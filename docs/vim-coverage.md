## エンジンポートの検証済み性質

- `reset()` 後は undo 履歴が前のお題に届かない(お題間の `u` 事故防止)
- キーストローク購読は Esc・モード切替キーも数える(R2)
- バッファ変更購読がクリア判定(R1)に使える粒度で発火する
- レジスタは reset をまたいで残る(本物の vim と同じ挙動として許容)
